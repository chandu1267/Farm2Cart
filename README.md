# 🌾 Farm2Cart — Frontend

## 📌 Overview
Farm2Cart is a B2B e-commerce platform for agricultural products.  
This repository contains the **frontend (client-side)** built using React.js, providing a responsive and user-friendly interface for buyers and sellers.

## 🛠 Tech Stack
- React.js
- Tailwind CSS / CSS3
- Axios
- React Router

## ✨ Features
- 🔍 Browse and search products
- 🛒 Add to cart & manage cart items
- 🔐 User authentication (OTP-based login)
- 📦 View product details with images
- ⚡ Lazy loading for better performance
- 📱 Fully responsive UI
